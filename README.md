# Semantic Profile Page

A simple, accessible profile page built with HTML5 semantic structure and CSS fundamentals.

## Features

- Semantic HTML5 page sections: `header`, `main`, `section`, `aside`, and `footer`
- External CSS for layout, typography, and responsive design
- Accessibility checklist to verify best practices
- Keyboard-friendly focus styling and readable contrast
- Clean, modern profile page structure

## Files

- `index.html` — profile page markup with semantic elements
- `styles.css` — external stylesheet for styling and responsive behavior
- `.gitignore` — ignores common system files

## Local usage

1. Open `index.html` in a browser.
2. Review the accessibility checklist section.
3. Adjust content and style as needed for your profile.

## Accessibility notes

- Uses meaningful `alt` text for the profile image
- Includes landmark regions for navigation
- Uses logical heading order
- Provides visible focus styles for interactive elements
- Uses lists for structured content and checklist items
